# Hardware
## 센서부
* RTK-GPS x 2   (SMC-2000, SMC-PLUS)
* 3D Lidar      (MID-360)
* RGB Camera    (ELP-USB-CAM)

## 추진부
* Thruster      (Bluerobotis T200)
* Motor Driver  (Bluerobotis Basic ESC)

## 제어부
* SBC   (Intel NUC)
* MCU   (Arduino Mega 2560)
* RadioMaster TX12 Controller

## 전기
* 24V Battery
* 12V Converter

## 안전
* Emergency Switch
* Relay
* Fan
* RGB NeoPixel LED


# Software
## 개발 환경
* Ubuntu 20.04
